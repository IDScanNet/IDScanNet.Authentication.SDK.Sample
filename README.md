# IDScanNet.Authentication.SDK.Sample

This is a sample application for demonstrating how to use the IDScan Authentication SDK 

## Usage

Execute Sample without arguments to run ASDK on test scan set.

Execute Sample with folder path as argument to run ASK on customer scan set.

Sample can handle files in scan set:
```
  Pdf417RawData.txt

  *ScanIrBack*.*
  *ScanIrFront*.*
  *ScanNormalBack*.*
  *ScanNormalFront*.*
  *ScanUvBack*.*
  *ScanUvFront*.*

  *Face*.*
```
